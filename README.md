# Severstal Notes

Небольшое одностраничное приложение для заметок на React + Tailwind CSS, которое хранит данные в `localStorage` и работает офлайн.

## Возможности

- Создание, удаление и редактирование заметок.
- Автосохранение в `localStorage` и восстановление при следующем запуске.
- Поиск по заголовку и тексту заметок.
- Сортировка по последнему обновлению.
- Встроенное форматирование текста в редакторе: жирный, курсив, подчёркнутый, выбор шрифта и размера.
- Уведомления о критических ошибках через toast и fallback-экран с возможностью сброса данных.

## Технологии

- **React 19** + **Vite (rolldown-vite)**
- **Tailwind CSS v4**
- **react-hot-toast** для уведомлений
- **Vitest** + **Testing Library** для тестов
- **Docker + Nginx** для production-сборки

## Быстрый старт

Установка зависимостей:

```bash
npm install
```

Запуск в режиме разработки:


## Что реализовано

- **Состояние заметок** хранится в `App`, изменения сохраняются в `localStorage` при любом обновлении.
- **Первый запуск** создаёт стартовую заметку с подсказками.
- **Список заметок** показывает превью текста (из HTML), активную заметку и кнопку удаления.
- **Редактор** использует `contentEditable`, содержит поле заголовка и панель форматирования (шрифт, размер, жирный/курсив/подчёркнутый).
- **Вставка изображений** в заметки через кнопку в панели, вставку из буфера или drag&drop; изображения сохраняются в HTML заметки.
- **Обработчик ошибок** (`AppErrorBoundary`) показывает экран ошибки, позволяет перезагрузить страницу и сбросить данные.
- **Тесты** покрывают первый запуск, создание/редактирование и сохранение между сессиями.
