# Severstal Notes

Небольшое одностраничное приложение для заметок на React + Tailwind CSS, которое хранит данные в `localStorage` и работает офлайн. Подходит как демонстрационный проект для Vite, React 19 и тестирования через Vitest + Testing Library.

## Возможности

- Создание, удаление и редактирование заметок.
- Автосохранение в `localStorage` и восстановление при следующем запуске.
- Поиск по заголовку и тексту заметок.
- Сортировка по последнему обновлению.
- Быстрые кнопки форматирования Markdown (жирный, курсив, моноширинный).
- Обработчик критических ошибок с toast-уведомлением и fallback-экраном.

## Технологии

- **React 19** + **Vite (rolldown-vite)**
- **Tailwind CSS v4**
- **react-hot-toast** для уведомлений
- **Vitest** + **Testing Library** для тестов
- **Docker + Nginx** для production-сборки

## Быстрый старт

Установка зависимостей:

```bash
npm install
```

Запуск в режиме разработки:

```bash
npm run dev
```

Сборка production-бандла:

```bash
npm run build
```

Предпросмотр production-сборки:

```bash
npm run preview
```

## Тесты и качество

Запуск тестов:

```bash
npm run test
```

CI-режим тестов:

```bash

```

Проверка линтером:


Форматирование кода:

Сборка и запуск контейнера:

```bash
docker compose up --build
```

Приложение будет доступно на `http://localhost:8080`.

## Структура проекта

```
src/
  app/                # корневой компонент приложения
  entities/note/      # доменная логика заметок (model, api, ui)
  shared/lib/         # общие утилиты (storage, error boundary)
  styles/             # глобальные стили (Tailwind)
  main.jsx            # точка входа
```

## Что реализовано

- **Состояние заметок** хранится в `App`, а изменения сохраняются в `localStorage` при любом обновлении состояния.
- **Первый запуск** создаёт стартовую заметку, которая подсказывает, как пользоваться приложением.
- **Список заметок** показывает превью текста, активную заметку и кнопку удаления.
- **Редактор** содержит поле заголовка, textarea и панель Markdown-команд (B, I, </>).
- **Обработчик ошибок** (`AppErrorBoundary`) показывает экран ошибки и позволяет перезагрузить приложение.
- **Тесты** покрывают создание, редактирование и сохранение заметок.
